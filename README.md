# Mini CRM

A Full Stack Lead Management System built using React, Node.js, Express, and PostgreSQL.

## Tech Stack

- React.js
- Node.js
- Express.js
- PostgreSQL

## Features

- Add Lead
- View Leads
- Update Lead Status
- Delete Lead
- Search Leads
- Dashboard Statistics
- Responsive UI

## Run Backend

```bash
cd backend
npm install
npm start
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
